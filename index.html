VSCO KEYS

NOW OPEN SOURCE

Free for All

Keys was originally distributed by Visual Supply Company in order to give users a speed boost when editing images in Adobe Photoshop Lightroom. In August 2015, it was discontinued as a product for purchase, and is now open source and available for free download.

VSCO has open-sourced the code for the installer and Keys app. While the current layout editor has been discontinued, users can continue to edit their layouts with a text editor using resources that have been provided in the source code. For more information, view VSCO's support article or the Keys README on GitHub.

Keys is a powerful keyboard shortcut tool created to drastically reduce image editing time in Adobe Lightroom CC, 6, 5, 4, and 3.
